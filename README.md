# Striproute Demo
*API’s are the new black.* Cities like Brussels provide open data, so in this DEMO a *comic route* API is used to get a list of characters. The main goals of this BRANCH are:
- filter data by using LinQ
- show a list of comic characters

This Branch is parrallel with step 3 and 4 `Specifieke gegevens filteren` & `Consoleapplicatie testen`.

## Requirements
- Visual Studio
- Newtonsoft Json.net

## Sources
API: [Open data Brussels] (http://opendata.brussels.be/explore/dataset/striproute0/)
